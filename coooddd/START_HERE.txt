🔌 ESP32 CONNECTION:

  WITHOUT ESP32:
  ──────────────
  Just open the dashboard! It automatically generates
  realistic solar data. Perfect for testing!
  
  WITH ESP32:
  ───────────
  1. Upload ESP32 code (see README.md for code)
  2. Connect ESP32 to WiFi
  3. Note the IP address from Serial Monitor
  4. Open Settings page in dashboard
  5. Enter ESP32 IP address
  6. Click "Save ESP32 Settings"
  
  Done! Real data starts flowing!
  
  Default Password: solar123

🆘 TROUBLESHOOTING:

  Dashboard not updating?
    → Refresh browser (Ctrl + F5)
    → Check browser console (F12)
  
  ESP32 not connecting?
    → Verify IP address correct
    → Check ESP32 on same WiFi network
    → Review ESP32 Serial Monitor
  
  Dark mode not working?
    → Clear browser cache
    → Check localStorage enabled
  
  Charts not showing?
    → Check internet connection (Chart.js CDN)
    → Try different browser
